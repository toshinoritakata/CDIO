# CGWORLD連載サンプル
CONTEC DIO-0808LY-USBを利用してUnityにボタンをつなげるサンプルです。

ボタン0を押すと移動、ボタン1を押すと回転が早くなります。

同時に押すと全体の移動量も増えます。

オブジェクトの動きの制御に https://github.com/keijiro/Klak の Brownian Motion を利用しています。
